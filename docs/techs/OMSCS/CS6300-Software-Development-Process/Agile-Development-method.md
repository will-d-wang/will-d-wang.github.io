# CS6300 SAD Agile Development Methods

## Cost of change

From exponential to flat.

## Agile Software Development

* Focus on the code
* People over process
* Iterative approach
* Customer involvement
* **Expectation that requirements will change**
* **Simplicity**.

* Lightweight

* Discipline
* Software Development

* Continuous Integration.
