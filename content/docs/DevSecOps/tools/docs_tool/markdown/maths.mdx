Markdown can support Latex grammer for symbol and math.

Greek Letters

| Symbol    | Markdown txt    | Latex         |
| --------- | --------------- | ------------- |
| $\alpha$  | ```$\alpha$```  | ```\alpha```  |
| $\beta$   | ```$\beta$```   | ```\beta```   |
| $\gamma$  | ```$\gamma$```  | ```\gammma``` |
| $\Gamma$  | ```$\Gamma$```  | ```\Gammma``` |
| $\pi$     | ```$\pi$```     | ```\pi```     |
| $\Pi$     | ```$\Pi$```     | ```\Pi```     |
| $\phi$    | ```$\phi$```    | ```\phi```    |
| $\Phi$    | ```$\Phi$```    | ```\Phi```    |
| $\varphi$ | ```$\varphi$``` | ```\varphi``` |
| $\theta$  | ```$\theta$```  | ```\theta```  |
| $\Theta$  | ```$\Theta$```  | ```\Theta```  |
