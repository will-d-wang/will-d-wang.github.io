Welcome to my corner of the internet! Here you'll find my notes on technology and computer science.

I've documented my journey of continuous learning as a software developer, inspired by [The Ph.D. Grind: A Ph.D. Student Memoir](https://www.goodreads.com/en/book/show/15731248-the-ph-d-grind).

Great software engineers read broadly, master complex technologies, and strive to deliver elegant, concise code in their work.

The software engineering path is a lifelong pursuit of [craftsmanship](https://manifesto.softwarecraftsmanship.org/).

Alongside traditional software engineering topics, these docs also include my notes on prompt engineering and insights learned while interacting with AI (like ChatGPT) about computer science and technology.
